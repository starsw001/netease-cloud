# 项目结构



## netease-cloud

```
|-- 项目文件夹
    |-- LICENSE
    |-- README.md
    |-- account.json
    |-- init.config
    |-- main.py
    |-- requirements.txt
    |-- run.log
```

`LICENSE`：开源许可证

`README.md`：项目自述文件

`account.json`：账号存放文件

`init.config`：配置文件

`main.py`：主程序

`requirements.txt`：依赖清单

`run.log`：运行日志
